# Login API

Projeto de API de Login desenvolvido em Java com Spring Boot, utilizando Clean Architecture.

## Estrutura de Módulos

*   **domain**: Contém as classes pertencentes ao core e serviços de aplicação, com regras de negócio blindadas (.
*   **springframework**: Contém as configurações do Spring Boot, injeção de dependências e os componentes de infraestrutura (Controladores REST e Repositórios).

## Tecnologias

*   Java 17
*   Spring Boot 3
*   MongoDB
*   Lombok
*   JUnit
